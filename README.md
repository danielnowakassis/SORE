### Repository for the paper : Self Optimizing Regularized Ensemble (SORE)

## Configuration of the MOA framework : [link](https://github.com/Waikato/moa)

- Files in ['classifiers'](classifiers/) contain what should be added to the MOA, for meta and trees, just add (or replace the files), and add the entire HPO folder to moa/classifiers

## Datasets

- Synthetic datasets generation for the MOA framework can be found in the [`generate_synth.py`](generate_synth.py)
- Real-World data is available in [link](https://sites.google.com/view/uspdsrepository)
- For CaDrift, the dataset is in [link](https://drive.google.com/drive/folders/1Cboa-f7Nex3MC9FWkkakztuOcJFZ9uZJ?usp=sharing)

## Run

- After configuring the MOA framework, compress it in a .jar file, you are able to execute the [`run.py`](run.py) file
